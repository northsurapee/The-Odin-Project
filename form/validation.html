Validation

Validations allow us to set specific constraints or rules that determine what data users can enter into an input.

Required validation (add attribute 'required' to input tag)


Text length

Minlength (add 'minlength')

Maxlength (add 'maxlength')

Number range

The min and max attributes only work with number-based form controls such as the number, dates and time inputs.

Minvalue

Maxvalue

Pattern Validation

To add a pattern validation, we give the form control a pattern attribute with a regular expression as the value.

Styling validations

We can target form controls that have passed or failed validations using the :valid and :invalid pseudo-classes.
